INPUT FOR CLIENT FILE :

1. Enter the implementation - 0 for Array and 1 for Linked List.
2. Enter the Capacity for the Array / Linked List.
3. INSERT FUNCTION - Enter the key and the value pair.
4. SEARCH FUNCTION - Enter the key to search.
5. DELETE FUNCTION - Enter the key to be deleted.

COMMAND TO BE EXECUTED IN THE TERMAINAL :

1. make -f make.mk
2. ./a.out
